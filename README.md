## Description
This project is a Machine Learning-based system that predicts the possibility of chronic diseases using patient health data.

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization (heatmaps, histograms, graphs)
- Machine Learning model implementation
- Prediction of chronic disease

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset
The dataset includes:
- Age
- Blood Pressure
- Sugar Level
- Other health-related features
